# Methodology

---

## Purpose of the Dataset
The dataset is used to demonstrate data cleaning and visualization techniques. It contains missing values and simple numerical features for analysis.

---

## Handling Missing Values
Missing values were handled using **mean imputation**, where missing entries are replaced with the average of the respective column.

---

## Why This Strategy Was Chosen
Mean imputation is simple, fast, and effective for small datasets. It helps maintain dataset size without removing rows.

---

## Visualizations Generated
The following visualizations were created:
- Bar plots  
- Histograms  
- Basic statistical summaries  

These help in understanding data distribution and trends.

---

## Limitations
- Mean imputation can introduce bias  
- Not suitable for large or complex datasets  
- Limited visualization techniques used
