# Reproducibility Checklist

---

## Input Files Required
- Raw dataset in `data/` folder  

---

## Scripts to be Executed
- src/data_cleaning.py  
- src/visualization.py  

---

## Execution Order
1. Run data_cleaning.py  
2. Run visualization.py  

---

## Expected Output Files
- Cleaned dataset in `data/`  
- Plots in `results/`  

---

## Software Dependencies
- Python 3.x  
- pandas  
- matplotlib  
- numpy  

---

## Assumptions
- Dataset is cleanable using simple techniques  
- Missing values are minimal  

---

## Limitations
- Limited dataset size  
- Basic preprocessing techniques
